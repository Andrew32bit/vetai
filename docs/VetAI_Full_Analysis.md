## 2. ФИКСАЦИЯ ИДЕИ ПРОДУКТА

### Название: VetAI

### Слоган: «AI-ветеринар в кармане»

### Миссия
Сделать ветеринарную диагностику доступной для каждого владельца животного — от хозяина кошки в Москве до фермера с 50 коровами в Саратове.

### Что это
Telegram Mini App + бот, который по фото, загруженным анализам и описанию симптомов даёт предварительную AI-оценку здоровья животного с рекомендациями.

### Для кого (Фаза 1 — B2C)
Владельцы домашних животных в России (57 млн кошек + 23 млн собак), которые хотят быстро понять: «это серьёзно или нет?» — до визита к ветеринару.

### Для кого (Фаза 2 — B2B)
Малые и средние фермы (97% всех КРС-предприятий), которым нужен мониторинг здоровья стада, но крупные решения неподъёмны.

### Ключевое отличие от всех конкурентов
**3-в-1 в одном Telegram-приложении:**
1. **Фото → AI-диагноз** (как TTcare, но на русском + больше категорий)
2. **Анализы → расшифровка** (никто в мире не делает это для животных в массовом B2C)
3. **Чат → AI-опрос** (как Vet-AI, но с фото и анализами)

### Технологический стек
- **CV-модель:** EfficientNet-B4 / YOLOv8 fine-tuned на ветеринарных датасетах
- **OCR:** Tesseract / PaddleOCR для распознавания бланков анализов
- **LLM:** Claude API для чат-диагностики и генерации рекомендаций
- **Backend:** Python + FastAPI
- **Frontend:** Telegram Mini App (React)
- **Инфра:** Yandex Cloud

```
        → Нажимает ссылку → Открывается VetAI Mini App

Шаг 2: Быстрая регистрация (30 сек)
        → Имя: Мурка
        → Вид: Кошка
        → Порода: Британская
        → Возраст: 3 года
        → Вес: 4.5 кг

Шаг 3: Главный экран → «Что беспокоит?»
        → Маша нажимает «📸 Загрузить фото»

        → Загружает в приложение

        → CV-модель: «Аллергический дерматит, уверенность 78%»

Шаг 6: Получает рекомендации:
        1. Исключить новые продукты из рациона
        2. Визит к ветеринару в течение 48 часов

        → Кнопка «Найти ветклинику рядом»

Шаг 7: Маша записывается в клинику через приложение
        → VetAI получает 300₽ комиссии за лид
```

### Сценарий Б: «Расшифровать анализ крови»

**Персона:** Игорь, 45 лет, Нижний Новгород, собака Рекс (7 лет, немецкая овчарка)

```
Шаг 1: Игорь забрал анализы из лаборатории (PDF на email)

Шаг 2: Открывает VetAI → «📋 Загрузить анализы»

Шаг 3: Прикрепляет PDF «analiz_krovi_reks.pdf»

Шаг 4: AI обрабатывает (5–10 секунд):
        → OCR извлекает значения
        → Сравнивает с нормами для немецкой овчарки, 7 лет

          ✓ Гемоглобин — 128 г/л (норма)
          ↑ Лейкоциты — 18.5 (повышены → возможно воспаление)
          ✓ Тромбоциты — 310 (норма)
          ↑ АЛТ — 85 (незначительно повышен → контроль печени)

Шаг 5: Простым языком:
        «Повышенные лейкоциты могут говорить о воспалительном
        процессе. АЛТ чуть выше нормы — рекомендуется пересдать
        через 2 недели. Это не экстренная ситуация, но покажите
        результаты вашему ветеринару на следующем визите.»

Шаг 6: Игорь сохраняет отчёт в историю
        → Через 2 недели получает напоминание пересдать анализ
```

## 4. МАРКЕТИНГОВАЯ СТРАТЕГИЯ

### Почему Telegram

- 85+ млн пользователей в РФ (март 2026)
- Mini App — нативный формат, не нужна установка из магазина
- Реклама в Mini Apps: CTR на 40% выше обычных каналов
- Виральность: легко переслать бота другу
- Стоимость подписчика: 25–50₽ (vs 100–200₽ в мобильных приложениях)
