**Механика:** «Пригласи друга — получите по 5 дополнительных проверок»

**Как работает:**

1. У каждого пользователя есть уникальная реферальная ссылка:
   `t.me/vetai_bot?start=ref_{user_id}`

2. Когда новый пользователь переходит по ссылке:
   - Новый пользователь получает +5 бесплатных проверок (сверх стандартных 10/день)
   - Пригласивший получает +5 бесплатных проверок
   - Оба получают уведомление

3. Лимиты:
   - Максимум 10 приглашений на пользователя (50 бонусных проверок)
   - Бонусные проверки не сгорают

4. Геймификация:
   - 3 приглашения — бейдж «Друг питомцев»
   - 5 приглашений — бейдж «Защитник здоровья»
   - 10 приглашений — 1 месяц подписки бесплатно

**Техническая реализация:**

```
Backend: добавить поле referral_code и referred_by в модель User
Эндпоинт: при /start с параметром ref_{id} — записать referred_by, начислить бонусы обоим
Уведомление: бот отправляет сообщение пригласившему: «Ваш друг присоединился! +5 проверок»
```
