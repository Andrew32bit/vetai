## 3. vc.ru (IT-аудитория + виральность)

### Текст статьи

**Заголовок:** Как я сделал AI-ветеринара в Telegram за 2 недели: стек, промпты, грабли

```
Я — разработчик, и у меня есть кот. История началась в 3 часа ночи, когда он начал хромать...

[Рассказ о проблеме и мотивации]

Стек:
— Backend: Python + FastAPI
— AI: Groq (Llama 70B) для чата + Claude Vision для фото
— Frontend: React + Telegram Mini App SDK
— Хостинг: Yandex Cloud

Что умеет бот:
1. Анализ фото — загружаете фото питомца, AI определяет возможное заболевание
2. Расшифровка анализов — фото бланка → таблица показателей с отклонениями
3. Чат по симптомам — описываете что происходит, бот оценивает срочность

Интересные технические решения:
— Fallback Groq → Claude при rate limit (бесшовное переключение)
— Промпт-инжиниринг для точной диагностики по фото
— Защита от jailbreak (тест Гендальфа — 5/5)

Юнит-экономика:
— Groq: бесплатно (110 запросов/день)
— Claude fallback: ~$0.009/запрос
— Первые 22 пользователя/день — полностью бесплатно

Попробовать: @vetai_app_bot (Telegram)

Буду рад фидбеку от сообщества!
```
